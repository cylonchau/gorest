# restclient
Transplanted from kubernetes client-go's rest client http client

### Features

- HTTP Authtication
- Timeout
- Failer retry
- Cancel(with context)
- HTTP Proxy
- Native go net.http package
- Rest Style

### Installation

```
go get github.com/cylonchau/restclient
```
